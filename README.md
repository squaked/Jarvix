# Jarvix — Your AI assistant for Mac

---

## Install

Open **Terminal** (press `⌘ Space`, type "Terminal", press Enter) and paste:

```bash
curl -fsSL https://raw.githubusercontent.com/squaked/Jarvix/main/install.sh | bash
```

If you do **not** have brew already installed, paste:
This will ask you for your **mac login** password.

```bash
sudo curl -fsSL https://raw.githubusercontent.com/squaked/Jarvix/main/install.sh | bash
```

## Open

Press `⌘ Space`, type **Jarvix**, press Enter — or click the Jarvix icon in your Dock.

**Manual update**
Use if something is broken, after quitting Jarvix, run the installer again.

**Uninstall**

```bash
curl -fsSL https://raw.githubusercontent.com/squaked/Jarvix/main/scripts/uninstall.sh | bash
```
